# Точка Пая — сайт сервисного центра

## Запуск

```bash
npm install
npm start
```

Откройте в браузере: **http://localhost:3000**

## База данных (SQLite)

Файл: `db/tochka_paya.db`

### Таблицы

| Таблица | Назначение |
|---------|------------|
| `services` | Услуги и цены на ремонт |
| `products` | Запчасти в магазине |
| `repair_orders` | Заявки и заказы на ремонт |
| `order_history` | История статусов заказа |
| `courier_requests` | Заявки на вызов курьера |
| `shop_orders` | Заказы из магазина запчастей |

### Демо-заказ для проверки статуса

Номер: **ТП-2026-0042**

## API

- `GET /api/services` — список услуг
- `GET /api/products` — каталог запчастей
- `POST /api/orders` — новая заявка на ремонт
- `GET /api/orders/:номер` — статус ремонта
- `POST /api/courier` — вызов курьера
- `POST /api/shop-orders` — оформление заказа из корзины
